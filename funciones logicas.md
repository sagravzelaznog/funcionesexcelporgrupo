Funciones Lógicas más Usadas en Excel
Funciones Condicionales y de Decisión
• SI: Es la más importante. Evalúa una condición y devuelve un valor si es verdadera y otro si es falsa.

• SI.ERROR: Evalúa una fórmula y devuelve un valor alternativo si se produce un error (como #N/D o #¡DIV/0!); si no hay error, devuelve el resultado normal.

• SI.CONJUNTO: Comprueba una o varias condiciones y devuelve un valor correspondiente a la primera condición que sea VERDADERA. Evita anidar múltiples "SI".

• SI.ND (o SI.NA): Similar a SI.ERROR, pero solo detecta el error #N/D (valor no disponible).

Operadores Lógicos (Comúnmente usados dentro de SI)
• Y: Comprueba si todas las condiciones son VERDADERO.

• O: Comprueba si al menos una de las condiciones es VERDADERO.

• NO: Invierte el valor lógico de su argumento (cambia FALSO a VERDADERO y viceversa).

• XO: Devuelve un "O exclusivo" lógico de todos los argumentos (VERDADERO solo si un número impar de argumentos es verdadero).

Valores Lógicos Directos
• VERDADERO: Devuelve el valor lógico VERDADERO.

• FALSO: Devuelve el valor lógico FALSO.

Nuevas Funciones (Office 365 / Excel Reciente)
• CAMBIAR: Evalúa una expresión comparándola con una lista de valores y devuelve el resultado correspondiente al primer valor coincidente.

• LET: Permite asignar nombres a los resultados de los cálculos dentro de una fórmula para mejorar el rendimiento y la lectura.

• LAMBDA: Permite crear funciones personalizadas reutilizables.
