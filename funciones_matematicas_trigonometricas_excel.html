Funciones Matemáticas y Trigonométricas en Excel

Herramientas para cálculos matemáticos y trigonométricos

Matemáticas Básicas

▼

Funciones para operaciones matemáticas fundamentales.

Ejemplo 1: Funciones de suma

// Suma simple
=SUMA(A1:A10)  // Suma los valores del rango A1:A10

// Suma con condición
=SUMAR.SI(B1:B10, ">100")  // Suma valores mayores a 100

// Suma con múltiples condiciones
=SUMAR.SI.CONJUNTO(C1:C10, A1:A10, "=Rojo", B1:B10, ">50")

Ejemplo 2: Operaciones básicas

// Producto de un rango
=PRODUCTO(A1:A5)  // Multiplica todos los valores en A1:A5

// División entera y residuo
=COCIENTE(10, 3)  // Resultado: 3 (parte entera de 10/3)
=RESIDUO(10, 3)   // Resultado: 1 (resto de 10/3)

// Potencia y raíz
=POTENCIA(2, 3)   // 2 elevado a 3 = 8
=RAIZ(16)         // Raíz cuadrada de 16 = 4

Ejemplo 3: Números aleatorios

// Número aleatorio entre 0 y 1
=ALEATORIO()

// Número entero aleatorio entre 1 y 100
=ALEATORIO.ENTRE(1, 100)

// Para forzar la actualización de números aleatorios
// Presiona F9 o ve a Fórmulas > Calcular Ahora

Ejercicio 1: Cálculo de promedios ponderados

Datos: A1:A5 = Notas (80, 90, 75, 85, 95), B1:B5 = Pesos (0.2, 0.2, 0.15, 0.2, 0.25)

Calcula el promedio ponderado de las notas.

=SUMAPRODUCTO(A1:A5, B1:B5)/SUMA(B1:B5)
// Resultado: 85.75

Explicación: Multiplica cada nota por su peso, suma los resultados y divide entre la suma de los pesos.

Ejercicio 2: Números aleatorios únicos

Crea una fórmula que genere 5 números aleatorios únicos entre 1 y 50.

En Excel 365 con funciones dinámicas:

=SORTBY(ALEATORIO.ENTRE(1, 50, 5), ALEATORIO.ARRAY(5))

En versiones anteriores, usa una columna auxiliar con ALEATORIO() y ordena por ella.

Funciones de Redondeo

▼

Diferentes métodos para redondear números en Excel.

Ejemplo 1: Redondeo estándar

// Redondear a 2 decimales
=REDONDEAR(3.14159, 2)  // 3.14

// Redondear al entero más cercano
=REDONDEAR(3.7, 0)      // 4

// Redondear a la centena más cercana
=REDONDEAR(1234, -2)    // 1200

Ejemplo 2: Redondeo hacia arriba/abajo

// Redondear siempre hacia arriba
=REDONDEAR.MAS(3.2, 0)   // 4
=REDONDEAR.MAS(-3.2, 0)  // -4 (más negativo)

// Redondear siempre hacia abajo
=REDONDEAR.MENOS(3.9, 0)  // 3
=REDONDEAR.MENOS(-3.9, 0) // -3 (menos negativo)

// Entero inferior
=ENTERO(3.9)  // 3
=ENTERO(-3.9) // -4 (siempre hacia abajo en la recta numérica)

// Truncar decimales
=TRUNCAR(3.987, 1)  // 3.9 (elimina sin redondear)

Ejercicio 1: Redondeo comercial

Datos: Precios en A1:A5 = (12.345, 45.678, 78.901, 23.456, 56.789)

Redondea los precios a 2 decimales, pero si el tercer decimal es 5 o más, redondea hacia arriba.

=REDONDEAR(A1, 2)  // Para cada celda
// Resultados: 12.35, 45.68, 78.90, 23.46, 56.79

Nota: La función REDONDEAR ya sigue esta lógica por defecto.

Ejercicio 2: Redondeo a múltiplos

Redondea el número 123.456 al múltiplo de 5 más cercano (hacia arriba).

=REDONDEAR.MAS(123.456/5, 0)*5  // Resultado: 125

Explicación: Divide entre 5, redondea al entero superior, y multiplica por 5.

Funciones Trigonométricas

▼

Funciones para cálculos trigonométricos en radianes y grados.

Ejemplo 1: Funciones básicas

// Ángulo de 30 grados (convertir a radianes primero)
=SENO(RADIANES(30))  // 0.5

// Coseno de 60 grados
=COS(RADIANES(60))   // 0.5

// Tangente de 45 grados
=TAN(RADIANES(45))   // 1

Ejemplo 2: Funciones inversas

// Arcoseno de 0.5 (en radianes)
=ASENO(0.5)          // 0.5236 radianes

// Convertir a grados
=GRADOS(ASENO(0.5))  // 30 grados

// Arcotangente de 1
=GRADOS(ATAN(1))     // 45 grados

Ejemplo 3: Aplicaciones prácticas

// Calcular la hipotenusa de un triángulo (a=3, b=4)
=RAIZ(3^2 + 4^2)     // 5

// Convertir coordenadas polares a cartesianas (r=5, θ=30°)
// x = r * cos(θ)
=5 * COS(RADIANES(30))  // 4.33
// y = r * sen(θ)
=5 * SENO(RADIANES(30)) // 2.5

Ejercicio 1: Cálculo de altura

Estás a 20 metros de la base de un edificio y el ángulo de elevación a la parte superior es de 60°.

Calcula la altura del edificio.

// altura = distancia * tan(ángulo)
=20 * TAN(RADIANES(60))  // 34.64 metros

Explicación: La tangente del ángulo es igual al cateto opuesto (altura) entre el cateto adyacente (distancia).

Ejercicio 2: Frecuencia de ondas

Una onda tiene un período de 0.02 segundos. Calcula su frecuencia en Hz.

// frecuencia = 1 / período
=1/0.02  // 50 Hz

Nota: Este es un ejemplo de aplicación de matemáticas básicas en física.
