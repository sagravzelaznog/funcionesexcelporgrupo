Funciones de Información más Usadas en Excel
Validación de Contenido (Funciones "ES")
Estas funciones devuelven VERDADERO o FALSO dependiendo del contenido de la celda.

• ESBLANCO: Comprueba si una celda está vacía.

• ESNUMERO: Comprueba si un valor es un número.

• ESTEXTO: Comprueba si un valor es texto.

• ESLOGICO: Comprueba si un valor es lógico (VERDADERO o FALSO).

• ESFORMULA: Comprueba si una celda contiene una fórmula (útil para auditoría).

Detección de Errores
• ESERROR: Comprueba si un valor es un error (cualquiera, incluido #N/D).

• ESERR: Comprueba si un valor es un error (excepto #N/D).

• ES.NOD (o ESNA): Comprueba si un valor es el error #N/D (no disponible).

Información del Sistema y Formato
• CELDA: Devuelve información sobre el formato, la ubicación o el contenido de una celda (ej. color, ancho, tipo).

• INFO: Devuelve información sobre el entorno operativo actual (ej. versión de Excel, sistema operativo).

• TIPO: Devuelve un número entero que representa el tipo de datos de un valor (1=número, 2=texto, etc.).

• FORMULATEXTO: Devuelve la fórmula de una celda como una cadena de texto.

• ES.PAR: Devuelve VERDADERO si el número es par.

• ES.IMPAR: Devuelve VERDADERO si el número es impar.

• HOJA: Devuelve el número de la hoja a la que se hace referencia.

• HOJAS: Devuelve la cantidad de hojas en una referencia.
