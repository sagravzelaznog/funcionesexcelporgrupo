Conversión de Unidades y Sistemas Numéricos

▼

Funciones para convertir entre diferentes sistemas de medición y numeración.

Ejemplo 1: CONVERTIR unidades

// Convertir 100 millas a kilómetros
=CONVERTIR(100, "mi", "km")  // 160.9344 km

// Convertir 25 grados Celsius a Fahrenheit
=CONVERTIR(25, "C", "F")     // 77°F

// Convertir 1 año luz a kilómetros
=CONVERTIR(1, "ly", "m")/1000  // 9.46073E+12 km

Ejemplo 2: Conversión de bases numéricas

// Binario a decimal
=BIN.A.DEC("1010")     // 10

// Decimal a hexadecimal
=DEC.A.HEX(255)         // FF

// Octal a decimal
=OCT.A.DEC("77")        // 63

// Hexadecimal a binario
=HEX.A.BIN("A5")        // 10100101

Ejercicio 1: Conversión de unidades compuestas

Un automóvil consume 7 litros de gasolina cada 100 km. ¿Cuántas millas por galón (mpg) recorre?

Dato: 1 milla = 1.60934 km, 1 galón = 3.78541 litros

// 100 km = 100/1.60934 millas = 62.1371 millas
// 7 litros = 7/3.78541 galones = 1.8492 galones
// mpg = 62.1371 / 1.8492 = 33.6 mpg

=CONVERTIR(100, "km", "mi") / CONVERTIR(7, "l", "gal")  // 33.6 mpg

Números Complejos

▼

Funciones para trabajar con números complejos (en formato "a+bi" o "a+bj").

Ejemplo 1: Operaciones básicas

// Crear números complejos
=COMPLEJO(3, 4)        // 3+4i
=COMPLEJO(1, -2, "j")   // 1-2j

// Suma y resta
=IM.SUMA("3+4i", "1-2i")    // 4+2i
=IM.RESTA("5+3i", "2+1i")   // 3+2i

// Multiplicación y división
=IM.PRODUCTO("1+2i", "3+4i")  // -5+10i
=IM.DIV("1+2i", "3+4i")       // 0.44+0.08i

Ejemplo 2: Funciones avanzadas

// Módulo (magnitud)
=IM.ABS("3+4i")       // 5

// Argumento (ángulo en radianes)
=IM.ANGULO("1+1i")    // 0.7854 (π/4 radianes o 45°)

// Conjugado complejo
=IM.CONJUGADA("2-3i") // 2+3i

Ejercicio 2: Circuitos eléctricos

En un circuito de corriente alterna, la impedancia de un inductor es Z = jωL, donde ω=2πf.

Calcula la impedancia de un inductor de 10 mH a una frecuencia de 50 Hz.

// ω = 2πf = 2*PI()*50 = 314.16 rad/s
// Z = jωL = j*314.16*0.01 = j3.1416 Ω

=COMPLEJO(0, 2*PI()*50*0.01)  // 0+3.1416i

Funciones Especiales

▼

Funciones matemáticas avanzadas para aplicaciones de ingeniería.

Ejemplo 1: Funciones de Bessel

// Función de Bessel de primera especie, orden 0
=BESSELJ(2, 0)  // J₀(2) ≈ 0.2239

// Función de Bessel modificada, orden 1
=BESSELI(1.5, 1)  // I₁(1.5) ≈ 0.9817

Nota: Las funciones de Bessel son útiles en problemas de conducción de calor, ondas electromagnéticas y mecánica cuántica.

Ejemplo 2: Funciones de error

// Función de error
=FUN.ERROR(1)  // 0.8427

// Función de error complementaria
=FUN.ERROR.COMPL(1)  // 0.1573

// Relación: FUN.ERROR(x) + FUN.ERROR.COMPL(x) = 1

Ejemplo 3: Operaciones a nivel de bits

// Operación AND a nivel de bits (5 = 0101, 3 = 0011)
=BIT.Y(5, 3)  // 1 (0001)

// Desplazamiento a la izquierda (5 << 2)
=BIT.DESPLIZQDA(5, 2)  // 20 (0101 → 10100)

// Desplazamiento a la derecha (20 >> 2)
=BIT.DESPLDCHA(20, 2)  // 5 (10100 → 00101)

Ejercicio 3: Probabilidad normal

Calcula la probabilidad de que una variable normal estándar esté entre -1.96 y 1.96.

// P(-1.96 < Z < 1.96) = FUN.ERROR(1.96/√2) - FUN.ERROR(-1.96/√2)
=2*FUN.ERROR(1.96/RAIZ(2))-1  // 0.95 o 95%

Esto muestra que aproximadamente el 95% de los datos en una distribución normal está dentro de ±1.96 desviaciones estándar.
