Funciones Estadísticas en Excel

Aprende a utilizar las funciones estadísticas más útiles en Excel

Medidas de Tendencia Central

Estas funciones ayudan a encontrar el valor central o típico en un conjunto de datos.

Ejemplo 1: PROMEDIO, MEDIANA y MODA

Si en A1:A7 tienes las edades: 18, 19, 20, 20, 21, 22, 25:

=PROMEDIO(A1:A7)   // Resultado: 20.71 (suma de valores / cantidad)
=MEDIANA(A1:A7)   // Resultado: 20 (valor central)
=MODA(A1:A7)  // Resultado: 20 (valor más frecuente)

Ejemplo 2: MEDIA.GEOM

Si en B1:B4 tienes tasas de crecimiento: 1.1, 1.2, 1.15, 1.25:

=MEDIA.GEOM(B1:B4)  // Resultado: 1.173 (raíz n-ésima del producto)

Ejercicio 1: Análisis de calificaciones

Datos: A1:A10 contiene calificaciones: 85, 92, 78, 90, 85, 88, 95, 85, 90, 92

Calcula el promedio, la mediana y la moda de las calificaciones

=PROMEDIO(A1:A10)   // Resultado: 88.5
=MEDIANA(A1:A10)   // Resultado: 88.5
=MODA(A1:A10)  // Resultado: 85

Explicación: El promedio es la suma dividida entre 10. La mediana es el promedio de los dos valores centrales (88 y 89). La moda es 85 por ser el valor más frecuente.

Ejercicio 2: Crecimiento de inversión

Una inversión crece 10%, 15%, 8%, 12% y 10% en 5 años. ¿Cuál es la tasa de crecimiento anual promedio?

=MEDIA.GEOM(1.10, 1.15, 1.08, 1.12, 1.10) - 1  // Resultado: 10.93%

Explicación: La media geométrica es la forma correcta de promediar tasas de crecimiento compuesto.

Funciones de Conteo

Estas funciones ayudan a contar celdas según diferentes criterios.

Ejemplo 1: CONTAR vs CONTARA

Si en A1:A5 tienes: 10, "Texto", 20, "", 30:

=CONTAR(A1:A5)   // Resultado: 3 (solo números)
=CONTARA(A1:A5)  // Resultado: 4 (cualquier valor no vacío)
=CONTAR.BLANCO(A1:A5)  // Resultado: 1 (celda vacía)

Ejemplo 2: CONTAR.SI

Si en B1:B10 tienes calificaciones: A, B, A, C, B, A, A, D, B, A:

=CONTAR.SI(B1:B10, "A")     // Resultado: 5 (cuenta las A)
=CONTAR.SI(B1:B10, ">B")   // Resultado: 1 (solo A es mayor que B)

Ejercicio 1: Análisis de asistencia

Datos: A1:A30 contiene "Asistió" o "Faltó" para cada día del mes

Calcula el porcentaje de días con asistencia (A1:A30 tiene 22 "Asistió" y 8 "Faltó")

=CONTAR.SI(A1:A30, "Asistió")/CONTARA(A1:A30)  // Resultado: 73.33%

Explicación: Dividimos la cantidad de "Asistió" entre el total de días registrados.

Ejercicio 2: Conteo de productos por categoría

Datos: A1:A20 contiene categorías (Electrónicos, Ropa, Hogar, etc.)

Cuenta cuántos productos son de la categoría "Electrónicos"

=CONTAR.SI(A1:A20, "Electrónicos")

Explicación: La función cuenta cuántas veces aparece "Electrónicos" en el rango A1:A20.

Valores Extremos

Funciones para encontrar valores máximos, mínimos y sus posiciones.

Ejemplo 1: MAX y MIN

Si en A1:A10 tienes ventas: 1500, 2300, 1800, 2100, 1950, 2450, 1750, 2200, 2000, 1900:

=MAX(A1:A10)   // Resultado: 2450 (valor más alto)
=MIN(A1:A10)   // Resultado: 1500 (valor más bajo)
=K.ESIMO.MAYOR(A1:A10, 2)  // Resultado: 2300 (segundo valor más alto)

Ejemplo 2: JERARQUIA.EQV

Si en B1:B5 tienes: 85, 92, 78, 90, 88:

=JERARQUIA.EQV(B1, B$1:B$5)  // Para B1=85, resultado: 4 (cuarto lugar)

Ejercicio 1: Top 3 de ventas

Datos: A1:A20 contiene montos de ventas

Encuentra los 3 montos más altos de ventas

=K.ESIMO.MAYOR(A1:A20, 1)  // 1er lugar
=K.ESIMO.MAYOR(A1:A20, 2)  // 2do lugar
=K.ESIMO.MAYOR(A1:A20, 3)  // 3er lugar

Explicación: K.ESIMO.MAYOR devuelve el k-ésimo valor más grande del rango.

Ejercicio 2: Clasificación de estudiantes

Datos: A1:A30 contiene calificaciones de exámenes

¿En qué posición quedaría un estudiante con 92 puntos?

=JERARQUIA.EQV(92, A1:A30, 1)  // Devuelve la posición (1 = más alto)

Explicación: JERARQUIA.EQV devuelve la posición de un valor en un conjunto de datos, donde 1 es el valor más alto.

Medidas de Dispersión

Funciones para medir la variabilidad de los datos.

Ejemplo 1: DESVEST.M y DESVEST.P

Si en A1:A5 tienes: 9, 10, 11, 12, 13:

=DESVEST.M(A1:A5)  // Desviación estándar muestral: 1.58
=DESVEST.P(A1:A5)  // Desviación estándar poblacional: 1.41
=VAR.S(A1:A5)      // Varianza muestral: 2.5
=VAR.P(A1:A5)      // Varianza poblacional: 2.0

Ejercicio 1: Análisis de rendimiento

Datos: A1:A20 contiene tiempos de ejecución en segundos

Calcula la desviación estándar muestral de los tiempos

=DESVEST.M(A1:A20)

Explicación: Usamos DESVEST.M porque es una muestra de datos, no toda la población.

Ejercicio 2: Comparación de conjuntos de datos

Datos: A1:A10 (conjunto 1) y B1:B10 (conjunto 2)

¿Qué conjunto es más variable? Calcula el coeficiente de variación (CV) para cada uno

Para el conjunto 1 (A1:A10):
=DESVEST.M(A1:A10)/PROMEDIO(A1:A10)

Para el conjunto 2 (B1:B10):
=DESVEST.M(B1:B10)/PROMEDIO(B1:B10)

Explicación: El coeficiente de variación (desviación estándar/ media) permite comparar la variabilidad entre conjuntos con diferentes escalas.

Correlación y Predicción

Funciones para analizar relaciones entre variables y hacer predicciones.

Ejemplo 1: COEF.DE.CORREL

Si en A1:A5 tienes horas de estudio: 2, 4, 6, 8, 10

y en B1:B5 tienes calificaciones: 60, 70, 80, 85, 95:

=COEF.DE.CORREL(A1:A5, B1:B5)  // Resultado: 0.97 (alta correlación positiva)

Ejemplo 2: PRONOSTICO.LINEAL

Usando los mismos datos del ejemplo anterior:

=PRONOSTICO.LINEAL(12, B1:B5, A1:A5)  // Predice la calificación para 12 horas de estudio

Ejercicio 1: Análisis de correlación

Datos: A1:A20 = Gastos en publicidad, B1:B20 = Ventas

¿Existe una relación entre el gasto en publicidad y las ventas?

=COEF.DE.CORREL(A1:A20, B1:B20)

Explicación: Un valor cercano a 1 indica una fuerte correlación positiva, cercano a -1 una correlación negativa, y cercano a 0 indica poca o ninguna correlación.

Ejercicio 2: Predicción de ventas

Datos: A1:A12 = Meses (1-12), B1:B12 = Ventas mensuales

Predice las ventas para el mes 13 usando regresión lineal

=PRONOSTICO.LINEAL(13, B1:B12, A1:A12)

Explicación: La función ajusta una línea de tendencia lineal a los datos y predice el valor para x=13.
