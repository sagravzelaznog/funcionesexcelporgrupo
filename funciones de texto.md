Funciones de Texto más Usadas en Excel

Básicas
• CONCATENAR (o CONCAT): Une varios elementos de texto en uno solo.

• IZQUIERDA: Devuelve el primer carácter o caracteres de una cadena de texto.

• DERECHA: Devuelve el último carácter o caracteres de una cadena de texto.

• EXTRAE: Devuelve un número específico de caracteres desde una posición inicial.

• LARGO: Devuelve el número de caracteres de una cadena de texto.

Formato y Limpieza
• MAYUSC: Convierte el texto a mayúsculas.

• MINUSC: Convierte el texto a minúsculas.

• NOMPROPIO: Pone en mayúscula la primera letra de cada palabra.

• ESPACIOS: Elimina los espacios sobrantes del texto.

• LIMPIAR: Elimina caracteres no imprimibles.

Búsqueda y Reemplazo
• ENCONTRAR: Busca una cadena dentro de otra (distingue mayúsculas/minúsculas).

• HALLAR: Busca una cadena dentro de otra (no distingue mayúsculas/minúsculas).

• SUSTITUIR: Reemplaza texto existente con texto nuevo.

• REEMPLAZAR: Reemplaza caracteres en una posición específica.

Conversión
• TEXTO: Convierte un valor numérico a texto con un formato específico.

• VALOR: Convierte texto con formato de número a un valor numérico.
