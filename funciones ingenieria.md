Funciones de Ingeniería más Usadas en Excel
Conversión de Unidades y Sistemas Numéricos
Estas funciones son esenciales para transformar datos entre diferentes métricas y bases.

• CONVERTIR: Transforma un número de un sistema de medida a otro (ej. metros a pies, Celsius a Fahrenheit).

• BIN.A.DEC / DEC.A.BIN: Conversión entre binario y decimal.

• HEX.A.DEC / DEC.A.HEX: Conversión entre hexadecimal y decimal.

• OCT.A.DEC / DEC.A.OCT: Conversión entre octal y decimal.

Operaciones con Números Complejos
Excel maneja números complejos (formato "x + yi") mediante funciones específicas que empiezan con "IM".

• COMPLEJO: Crea un número complejo a partir de coeficientes reales e imaginarios.

• IM.ABS: Devuelve el módulo (magnitud) de un número complejo.

• IM.ANGULO: Devuelve el argumento (ángulo) de un número complejo en radianes.

• IM.SUMA / IM.RESTA: Suma o resta dos números complejos.

• IM.PRODUCTO / IM.DIV: Multiplica o divide números complejos.

• IM.CONJUGADA: Devuelve el conjugado complejo.

Funciones Especiales (Bessel y Error)
• BESSELJ / BESSELY: Funciones de Bessel (cilíndricas) de primera y segunda especie. Útiles en problemas de propagación de ondas.

• BESSELI / BESSELK: Funciones de Bessel modificadas.

• FUN.ERROR (ERF): Devuelve la función de error (integral de la distribución normal estándar), común en estadística y ecuaciones diferenciales.

• FUN.ERROR.COMPL (ERFC): Función de error complementaria.

Lógica de Bits
• BIT.Y / BIT.O / BIT.XO: Realizan operaciones lógicas (AND, OR, XOR) a nivel de bits entre dos números.

• BIT.DESPLIZQDA / BIT.DESPLDCHA: Desplaza los bits de un número a la izquierda o derecha.

Otras Funciones Útiles en Ingeniería
• DELTA: Comprueba si dos valores son iguales (devuelve 1 si lo son, 0 si no). Actúa como una función delta de Kronecker.

• MAYOR.O.IGUAL: Comprueba si un número es mayor o igual que un valor de umbral (devuelve 1 o 0). Útil para filtrado de señales o pasos.
